### 使用方法：

```shell
git clone https://github.com/Lixuannan/CodeLib-Local.git # 也可以是 https://gitee.com/lixuannan/CodeLib-Local.git
cd CodeLib-Local
pip3 install -r requirements.txt
python3 main.py
```

### Usage : 

```shell
git clone https://github.com/Lixuannan/CodeLib-Local.git # https://gitee.com/lixuannan/CodeLib-Local.git is also  able
cd CodeLib-Local
pip3 install -r requirements.txt
mv data.data.template data.data
python3 main.py
```
